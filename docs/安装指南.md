# 因果记忆 — 安装指南

## 环境要求

- Python 3.10+
- OpenClaw（或任何 Agent 系统）
- SiliconFlow API Key（用于向量嵌入，可选）

## 快速安装

```bash
# 克隆仓库
git clone https://github.com/MaiHHConnect/MHH-Causality-Memory.git
cd MHH-Causality-Memory

# 一键初始化
bash scripts/setup.sh
```

## 手动安装

### 1. 向量数据库初始化

```bash
cd scripts/gbrain
python3 init.py
```

### 2. 配置 OpenClaw memory_search

在 `openclaw.json` 中添加 gbrain 路径：

```json
{
  "memory": {
    "search": {
      "provider": "custom",
      "script": "/path/to/MHH-Causality-Memory/scripts/gbrain/search.py"
    }
  }
}
```

### 3. Wiki 结构

Wiki 四层目录已包含模板文件：

```
wiki/main/
├── events/           ← 事件层模板（直接使用或复制）
├── timeline/         ← 时间线层（按年/月/周自动生成）
├── relationships/    ← 关系链层
└── abstracts/       ← 抽象总结层
```

## 使用流程

1. **运行 setup.sh** 初始化环境和目录
2. **配置 OpenClaw** 的 memory_search 指向 `scripts/gbrain/search.py`
3. **每次 session 后**将事件沉淀到 `wiki/main/events/`
4. **向量索引自动更新**，下次搜索时自动联想

## 核心脚本说明

| 脚本 | 功能 |
|------|------|
| `search.py` | 向量搜索入口，被 memory_search 调用 |
| `ingest.py` | 批量导入 Wiki 内容到向量库 |
| `init.py` | 初始化 SQLite + 向量数据库 |
| `stats.py` | 查看向量库状态 |
| `setup.sh` | 一键初始化脚本 |
